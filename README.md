# Instruction for running server --

1. go into src directory
2. go into server directory
3. run ``` npm install ```
4. run ``` npm start ```

# Instruction for running App --

1. go into src directory
2. go into friendify directory
3. run ``` flutter pub get ```
4. run ``` flutter run  ```
